Recipe 16: Provisioning a Web Server using Terraform
Objective
Deploy a web server on an EC2 instance.
Assignment Task

Create an EC2 instance and configure Apache or Nginx web server.
Guidelines / Steps
1. Create Terraform configuration for EC2.
2. Configure Security Groups to allow:
o SSH (port 22)
o HTTP (port 80)
3. Add user_data script to install a web server.
4. Initialize Terraform.
5. Deploy the infrastructure.
6. Obtain the public IP address of the instance.
7. Access the web server using a browser.
Expected Outcome
A working web server accessible through a public IP.




main.tf:-

provider "aws" {
  region = "ap-south-1"
}

resource "aws_instance" "web" {

  ami           = "ami-048f4445314bcaa09"
  instance_type = "t3.micro"

  user_data = <<-EOF
              #!/bin/bash
              sudo yum update -y
              sudo yum install httpd -y
              sudo systemctl start httpd
              sudo systemctl enable httpd
              sudo sh -c 'echo "<h1>Hello From Terraform Web Server</h1>" > /var/www/html/index.html'
              EOF
}
